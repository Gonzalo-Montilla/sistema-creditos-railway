¿Cómo funcionan las rutas?

Las rutas agrupan barrios por zonas geográficas. Cuando se crea un crédito, el sistema asigna automáticamente el cobrador según el barrio del cliente.

Gestión de Cobradores

Un cobrador puede manejar múltiples rutas, y una ruta puede tener varios cobradores asignados para flexibilidad operativa.
